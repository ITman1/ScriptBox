JavaScript Engine DemosWindow object's dialogs

Global Window object basically defines 5 user prompts. The following types are currently supported:

void alert(optional DOMString message = ""); boolean confirm(optional DOMString message = ""); DOMString? prompt(optional DOMString message = "", optional DOMString default = "");
