JavaScript Engine DemosLocation interface

Global Location object contains property location that should implement the Location interface. This interface is fully supported, including its super interface URLUtils except the origin and searchParams attributes. It has been also partially implemented the URL interface that implements also the URLUtils interface.

Navigation is accomplished to the navigation algorithm according the specification. At this moment are supported following schemas: "http", "file", "javascript" and "https". Even if we can now handle only text/html content type, it has been implemented general mechanism for handling also such contents like are: plain text, multipart, media, plugin, XML, etc.

Location interface

void assign(DOMString url); void replace(DOMString url); void reload();

URLUtils interface

DOMString href; DOMString origin; DOMString protocol; DOMString username; DOMString password; DOMString host; DOMString hostname; DOMString port; DOMString pathname; DOMString search; URLSearchParams searchParams; DOMString hash;

URL interface

URL(DOMString url, optional DOMString base = "about:blank");
